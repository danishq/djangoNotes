Creating a New App


1. We create a new app by running command python manage.py startapp djangoApp
2. We configre settings.py and mention app name in INSTALLED_APPS array
3. we make aware main urls.py that we have a new app and transfer the url handling to that new app path('djangoApp/', include('djangoApp.urls'))
4. we create routing in urls.py of djangoApp
5. we make views.py in djangoApp and import necessary library and create function to render our A_app.html file.
6. We ensure the html file is located at correct place and note to remember that django automatically looks for file in template folder because we mentioned dirs['templates'] in settigs.py earlier.
